微信小程序标记语言wxml

1月 8, 2019

wxml

与html类似，是微信小程序组织页面结构的标记语言；

标签对照

准确的说，wxml中是组件，不是标签

小程序 html 块或者流 说明
view div
text span
button button
form
input
navigator
image
swiper 轮播图片
*

小程序必须的文件

  • app.json 必须在根目录，数据内容必须制定一个页面，也就说说pages数组需要一个元素；
  • project.config.json 项目配置说明文件
  • 页面文件，位置自己随意定义，一般在 pages/pagename/下包含四种文件 json、js、wxml、wxss，但只有js和wxml是必须的，
    js负责实例化一个page,wxml定义内容；
