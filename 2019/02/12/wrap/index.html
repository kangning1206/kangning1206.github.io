javascript 中的基本包装类型

2月 12, 2019

数据分类

先明白一个概念，JavaScript中数据类型分类基本类型和引用类型，其中基本类型数字、字符串、bool、null,undefined;

基本类型

基本类型特点表示的是一个简单的数据结构，其本身不提供任何方法或者属性；

比如字符串var str = 'hello'; 仅表示一段文字；

现实中

但实际使用生活中，我们又需要对字符串进行截取、搜索、转化大小写等需求，按照常理，这些需要必须使用引用对象才能完成，也就是说 var str = new String('hello') 方式，生成一个字符串对象，在使用其对象的方法；

JavaScript的基本包装类

JavaScript 设计者，为了简化步骤以及满足现实生活中，提出一个基本类型包装，使得我们在使用基本类型时，仍然可以使用引用对象的方法或属性；

var str = 'hello'; console.log(str.length) 打印5；

怎么做到的

JavaScript 引擎每当读取一个基本类型属性时，后台就会创建一个对应的基本包装类型对象，从而让我们可以能够调用此对象的一些方法，来操作这些数据；

分解

  1. 创建 Strging 类型实例；
  2. 在这个新实例中调用指定的方法；
  3. 销毁这个实例；

注意下，基本类型上，不能自定义设置属性或方法，只能使用对应的包装类属性和方法；

s1.name 返回undefined;

第二行代码给s1添加了name属性，但执行第三行s1.name时已经被销毁了，同时第三行在打印s1.name生成了新的包装类，而该对象又没有name属性；

总结：

  1. 基本类型就是简单的数据类型，不会有引用类型特性；
  2. 为了使用方便，所看到的方法或属性是基本中间包装过一次，经过处理的后结果；
  3. 优先使用基本类型，尽可能不使用基本类型对应的包装类去实例化，如new Sting()少使用；
